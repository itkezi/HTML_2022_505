[Text style]



1. 블록 레벨 텍스트 태그

: 혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시되는 태그
1) h태그 : heading태그 - 제목을 표시하는 태그로 h1 ~ h6이 있다.

가장 큰 제목태그

가장 작은 제목태그

2) p태그 : paragraph - 단락을 표시하는 태그로 줄바꿈 없이 텍스트가 한 줄로 표시됨
텍스트의 길이가 브러우저 창의 너비보다 길면 자동으로 줄이 바뀐다.

3) br태그 : 줄바꿈 (닫는 태그가 없다.)

4) hr태그(horizontal) : 주제가 바뀌거나 분위기 전환을 할 때 가로선 표시 (닫는 태그X)

5) div 태그(division) : 영역을 나누는 블록 태그

2. 인라인 레벨 텍스트 태그

한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠 만큼의 너비를 차지하는 태그

1) strong 태그 : 텍스트를 굵게 표시하는 인라인 레벨 태그 굵은 텍스트 (칸이 남았기 때문에 한줄로 됨)

실행에서 제외하고자 하는 코드는 주석처리를 한다.
한 줄을 한꺼번에 주석처리 : ctrl + shift + c (똑같이 누르면 해제 가능)
여러 줄을 한꺼번에 주석처리 : ctrl + shift + /
2) Span 태그 : 특별한 뜻은 없지만 텍스트의 일부만 스타일 적용하고자 할 때 사용

텍스트 일부만 스타일 적용하기


3. 목록을 만드는 태그

1) u1(unordered list) : 순서가 없는 목록 태그 list-style-type: disc; = 가장 기본타입 원형

  • 메뉴2
  • 메뉴3
  • 메뉴4
  • 메뉴5


2) ol(ordered list) : 순서가 있는 목록 태그
list-style-type : 의 기본타입은 demical(정수) 이다.
숫자, 알파벳 로마숫자로 순서를 표시한다.
start : 시작값 지정
type : 리스트 타입

  2. 메뉴2
  3. 메뉴3
  4. 메뉴4
  5. 메뉴5
  1. 메뉴1
  2. 메뉴2

4. 표를 만드는 태그

table 태그, tr(table row) 태그, td(table data)
행열로 이루어진 표의 형태를 만들 때 사용하는 태그
tr : 행에 해당하는 태그
td : 행 내부의 열에 해당하는 셀 태그
th(table head) : 행 내부의 제목에 해당하는 셀 태그
caption 태그 : 표에 사용하는 제목 태그 (중앙상단위치)
셀병합 : colspan="병합할 칸 수"
